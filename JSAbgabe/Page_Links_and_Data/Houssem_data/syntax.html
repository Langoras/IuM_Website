Syntax

Was ist Syntax?

Diese Seite erklärt die grundlegende Syntax und die wichtigsten Befehle in CSS.

Innerhalb dieser Regel legen Deklarationen mit Eigenschaften und Werten fest, wie die ausgewählten Elemente gestaltet werden.

Wie ist eine CSS-Regel aufgebaut?

Eine CSS-Regel besteht aus einem Selektor und einem Deklarationsblock.

p {
    color: red;
    font-size: 20px;
}

Der Selektor p wählt Absätze aus. Im Block stehen Eigenschaften wie color und font-size sowie deren Werte.

Wichtige Arten von Selektoren

  • Element-Selektor: z. B. p, h1
  • Klassenselektor: .box
  • ID-Selektor: #header

Diese Selektoren reichen für einfache Webseiten völlig aus.

Eigenschaften und Werte

  • color – Textfarbe
  • background-color – Hintergrundfarbe
  • font-size – Schriftgröße
  • margin – äußerer Abstand
  • padding – innerer Abstand
  • border – Rahmen

Kommentare in CSS

/* Das ist ein Kommentar */

Kommentare helfen, den Code übersichtlich und verständlich zu halten.

Typische Anfängerfehler

  • fehlendes Semikolon
  • vergessene geschweifte Klammern
  • Tippfehler bei Eigenschaften
  • falsche oder konkurrierende Selektoren

Zusammenfassung

CSS beschreibt das Aussehen von Webseiten. Regeln bestehen aus Selektoren und Eigenschaften. Mit CSS steuert man Farben, Abstände, Schriften und Layouts.

Weitere Beispiele für CSS

CSS bietet viele Möglichkeiten, Elemente auf einer Webseite zu gestalten. Ein wichtiger Teil davon ist das Box-Modell.

Mehr über Farben und Einheiten

Farben können als Namen, Hex-, RGB- oder HSL-Werte angegeben werden.

CSS in der Praxis

CSS wird meist in einer externen Datei eingebunden.
