Buttons mit CSS

Was sind Buttons und wofür werden sie genutzt?

Als Button wird jedes Element auf einer Website bezeichnet, welches auf einen Mausklick folgend eine Aktion ausführt. Zum Beispiel wird man zu einer neuen Unterseite weitergeleitet oder man bestätigt Transaktionen. Buttons können unterschiedlich aussehen und verschiedene Effekte haben. Diese wollen wir uns im folgenden Dokument anschauen.

Wie ist ein Button aufgebaut?

CSS-Eigenschaft Beschreibung
background-color Damit legen Sie die Hintergrundfarbe des selektierten HTML-Elemets fest.
border Mit border setzen Sie den Style für alle vier Rahmenseiten gleichzeitig. Die Eigenschaft ist eine Kurzschreibweise der Eigenschaften border-width, border-style und border-color.
color Damit setzen Sie die Textfarbe des selektierten HTML-Elementes.
padding Mit padding setzen Sie den Innenabstand (auch Polsterung genannt) zwischen dem Inhalt und dem Rahmen (border) für alle vier Seiten fest. Diese Eigenschaft ist eine Kurzschreibweise der Eigenschaften padding-top, padding-right, padding-bottom und padding-left. Der Standardwert ist 0 für alle vier Seiten. Geben Sie einen Wert an, gilt dieser Wert für den Innenabstand aller vier Seiten. Bei zwei Werten wird der erste für oben und unten und der zweite für links und rechts verwendet, und bei drei Werten ist der erste für oben, der zweite für links und rechts und der dritte für unten. Bei vier Werten geht es von oben mit Wert eins im Uhrzeigersinn weiter.
text-align Damit können Sie die horizontale Ausrichtung des eines HTML-Elements anpassen.
› left: Linksbündig ausgerichtet
› right: Rechtsbündig ausgerichtet
› center: Zentriert
› justify: Blocksatz, sodass ein Text rechts- sowie linksbündig ausgerichtet wird.
text-decoration Seit CSS 1 können Sie mit dieser Eigenschaft dem Text eine Dekoration wie Unter-, Über- oder Durchstrich hinzufügen. Der Standartwert ist "none".
› none: Keine Dekoration
› underline: Text unterstrichen
› overline: Text überstichen
› line-through: Text durchgestrichen
display Mit display bestimmen Sie wie ein HTML-Element auf der Seite dargestellt wird. Mit display: inline-block werden Elemente nebeneinander angezeigt.
font-size Mit dieser Eigenschaft stellen Sie die Größe der verwendeten Schrift ein. Die Standardgröße ist 16px. Die Schriftgröße kann als numerischer Wert oder als relativer Wert angegeben werden.
cursor Mit dieser Eigenschaft können Sie das Aussehen des Mauszeigers ändern, wenn dieser auf ein bestimmtes HTML-Element zeigt. Hierbei gibt es viele mögliche Werte, die angenommen werden können.

Formen

Ein Button kann verschiedene Fromen besitzen. Er kann sehr eckig sein mit spiten 90° Winkel oder auch recht rund sein. Dies beinflust aber nicht seine Funktionsweiße sonden nur ein Visueles aussehen was der endnurtzer zu sehen bekommt. dies wird mit dem Opperator "border-radius" angegeben. Bei 0px hat der Button spitze 90° Winkel. Bei einem "border-radius" von 9999px kommt eine perfekte rundung zustande.

Beispiele border-radius


    .button-border_0 {
    border-radius: 0px !important;
    }


    .button-border_1 {
    border-radius: 6px !important;
    }


    .button-border_2 {
    border-radius: 9999px !important;
    }

Farben

Bis her sind die Buttons noch alle sehr monochrom. Um dies zu ändern kann man den Buttons auch verscheiene tolle Farben geben. Dies könne sich sorgar verändern je nachdem was man mit dem Button machen. Ob man nun über ihn Hovert, Klickt oder er nur unberührt da ist. In diesen Abschnitt wird gezeigt wie man die Fraben bestimtm. Man kann die Fraben in drei methoden darstellen. Man kann:
1. denn Namen der Farbe hin schreiben z.b. color: Darkcyan
2. Oder man gibt den Hex-Code an der wäre in unserm Beispiel color: #008B8B wäre
3. Man kann auch die Farbe in R.G.B. hinschreibt was dann so aus sähe: color: (0,139,139)

Beispiele für Farben 


    .button-color_1 {
    backroudn-color: Darkcyan;
    }


    .button-color_2 {
    backroudn-color: #008B8B;
    }


    .button-color_3 {
    backroudn-color: rgb(0,139,139);
    }

Tipp: Diese Färbemethoden gelten auch für: background-color, color (Text), border-color, outline-color, box-shadow, fill und deren Varianten.

Größen

Man kann Buttons nicht nur in Form und Farbe verändern sondern auch die generele Größe des Buttons. In dieesen Abschnitt wird kurz und prägnat beschrieben wie sie die Größe ihres Buttons ändern können. Man kann die Größe einmal durch die "font-size" und einmal durch das "padding" bestimmen. Bei "font-size" gibt man in px an wie groß die schrift im innern des Buttons sein soll die größe beinflust. Mit "padding" bestimmt man wie groß der Abstand vom text zu den einzelnen gräzen ist. Angeben wird die mit px*px.

Besipiel für Größen 


    .button-font_10 {
    font-size: 10px;
    }


    .button-font_25 {
    font-size: 25px;
    }


    .button-font_50 {
    font-size: 50px;
    }
