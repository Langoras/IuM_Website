Textformatierung mit CSS

Was ist Textformatierung?

Textformatierung meint die Änderung der Darstellung von Text, um die Lesbarkeit zu verbessern und einen Text optisch ansprechender zu gestalten. Es gibt verschiedene Möglichkeiten, mit denen man das erreichen kann.
Zum Beispiel über die:
  • SchriftART
  • SchriftGRÖßE
  • SchriftFARBE
  • Absätze
  • Zeilenabstände
  • Hervorhebungen

Warum ist die Textgestaltung wichtig?

Eine übersichtliche und ansprechende Textgestaltung ist außerordentlich wichtig für den Aufbau einer Website. Sie beeinflusst neben der besseren Lesbarkeit zusätzlich auch, wie gut Informationen an den Leser vermittelt werden.
BEISPIEL:
D e r Z e i c h e n a b s t a n d z . B . s p i e l t o f f e n s i c h t l i c h e i n e w i c h t i g e R o l l e b e i d e r A u f n a h m e v o n I n f o r m a t i o n e n.

Schon kleine Anpassungen können die Wahrnehmung eines Textes deutlich verbessern. Ohne diese Gestaltungsmerkmale wirkt ein Text schnell unübersichtlich und ist anstrengend zu lesen. Die Textformatierung spielt also eine wichtige Rolle für ein gutes Nutzererlebnis auf einer Website.
Der folgende Abschnitt handelt davon, wie man mit CSS Texte formatiert. Anhand von Beispielen werden die verschiedenen Möglichkeiten der Gestaltung erklärt.

Umsetzung der Textformatierung

In dem Abschnitt Integration von CSS wurde bereits erarbeitet, dass es drei übliche Wege gibt um CSS in eine HTML-Datei einzubinden. Im Folgenden wird vorrausgesezt, dass mit einem externen Style-Sheet gearbeitet wird.
Bei der Formatierung von Text werden Eigenschaften in CSS genutzt, die gezielt das Aussehen/die optische Darstellung der Schrift eines Elements(??) verändern.
Dabei ist der Ablauf immer gleich:

Erst wird ein Element ausgewählt, welches optisch verändert werden soll, z.B. alle Sections (section), Überschriften (h1-h6) oder eine bestimmte Klasse wie ".box".
Danach wird festgelegt, wie der ausgewählte Text nachher aussehen soll. Hierfür gibt es eigene CSS-Eigenschaften wie zum Beispiel "font-size" auf die nachfolgend noch eingegangen wird. Diese Eigenschaften werden nachfolgend nur noch spezifiziert, indem individuelle Werte wie z.B. eine Schriftgröße ausgewählt wird.

Zusammenfassung

Unter Textformatierung versteht man alle Maßnahmen, die das Erscheinungsbild von Text beeinflussen.Ob die Wahl der Schriftart, der Größe, der Farbe oder der Abstände zwischen einzelnen Elementen. Eine gut abgestimmte Gestaltung trägt viel zur Lesbarkeit bei und hilft dabei, Inhalte klarer zu vermitteln. Bereits kleine Justierungen, zum Beispiel beim Zeilen- oder Zeichenabstand, können den Eindruck eines Textes deutlich verbessern.

Für die Umsetzung in CSS wird zuerst das HTML-Element ausgewählt, dessen Darstellung angepasst werden soll. Danach werden die passenden Eigenschaften mit individuellen Werten festgelegt. Diese Einstellungen werden in einem externen Stylesheet hinterlegt und gelten anschließend automatisch für alle ausgewählten Elemente.
Hier die wichtigsten Erkenntnisse auf einen Blick:
  • Wie ein Text formatiert ist trägt maßgeblich zur guten Lesbarkeit und somit zu einer interesannten Website bei.
  • Mithilfe von CSS kann ein Text sehr viel gestaltet werden.
  • font-family -> Schriftart
  • font-size -> Schriftgröße
  • color -> Schriftfarbe
  • margin -> Absätze & Außenabstände
  • line-height -> Zeilenabstände
  • font-weight, text-decoration oder font-style -> Hervorhebungen
