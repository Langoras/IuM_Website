Auf dieser Website finden Sie eine Einführung in das Thema CSS.
CSS ist die Abkürzung für Cascading Style Sheets und beschreibt eine Stylesheet-Sprache mit der das Aussehen von HTML-Dokumenten gestaltet werden kann.
Mit CSS können Sie beispielsweise Farben, Schriftarten, Abstände und Layouts von Webseiten festlegen.
