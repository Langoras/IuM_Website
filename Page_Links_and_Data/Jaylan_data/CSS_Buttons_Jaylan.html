Buttons mit CSS

Was sind Buttons und wofür nutzen wir sie.

Als Butten wir jedes Element auf einer Website bezeichnet welches auf einen Mausklick hin eine Aktion ausführt. Zum Beispiel wird man zu einer Neuen Unterseite weitergeleitet oder man bestätigt Transaktionen. Buttons können verscheiden aussehen und verschiedene Effekte haben diese wollen wir uns im folgenden Dokument anschauen.

Wie ist ein Button aufgebaut

Der Grundaufbau eines Buttons besteht aus mindestes neuen Elementen.
1. Die Farbe des Hintergrundes diese wird mir "background-color: [Farbe]" angeben.
2. Wie Rund bzw. eckig der Button sein soll. Angeben mit "border: [zahl]px".
3. Die Farbe welche für ein Farbwechsel genutzt werden soll gibt man mit "color: [Farbe]" an.
4. Das "padding" beschreibt wie weit die Kanten von dem text entfernt ist.
5. "text-align" gibt an wo im Button der Text steht.
6. Mit "text-decoration" ist es möglich den Text im Button zu unterstreichen zu färben oder auch einzurahmen.
7. "display: inline-block" sorgt dafür, dass der Button auf der Selben höhe ist wie alles andere in der Zeile auch.
8. Die "front-size" bestimmt wie groß die Schrift im Button ist.
9. Mit der "cursor" Eigenschaft bestimmt man wie sich der Mauszeiger verändern soll, wenn er über dem Buttom hovert.

Wie ist ein Button aufgebaut

<
CSS-Eigenschaft Beschreibung
background-color Damit legen Sie die Hintergrundfarbe des selekterten HTML-Elemets fest.
border Mit border seten Sie den Style für alle vier Rahmenseitern gleichzeitig. Die Eigenschaft ist eine Kurzschreibweise der Eigenschaften border-width, border-style und border-color.
color Damit setzen Sie die Textfarbe des selektierten HTML-Elementes.
padding Mit padding seten Sie den Innen abstand (auch Polsterung genannt) zwischen dem Inhalt und dem Rahmen (border) für alle vier Seiten fest. Diese Eigenschaft ist eine Kurzschreibweise der Eigenschaften padding-top, padding-right, padding-bottom und padding-left.
Der Standartwert ist 0 für alle vier Seiten. Gebn Sie einen Wert an, gilt dieser Wert für denn Innenabstand aller vier Seiten. Bei zwei Werten wird der erste für oben und unten und der zweite für links und rechts verwendet, und bei drei Werten ist der erste für oben, der zweite für links und rechts und der dritte für unten. Bei vier Werten geht es von oben mit Wert eins im Uhrzeigersinn weiter.
text-align Damit können Sie die horizontale Ausrichtung des eines HTML-Elements anpassen.
› left: Linksbündig ausgerichtet
› right: Rechtsbündig ausgerichtet
› center: Zentriert
› justify: Blocksatz, sodass ein Text rechts- sowie linksbündig ausgerichtet wird.
text-decoration Seit CSS 1 können Sie mit dieser Eigenschaft dem Text eine Dekoration wie Unter-, Über - oder Durchstrich hinzufügen. Der Standerdwert ist "none".
› none: Keine Dekoration
› underline: Text unterstrich
› overline: Text überstichen
› line-through: Text durchgestrichen
display Mit display bestimmen Sie wie ein HTML-Element auf der Seite dargestellt wird. Mit display: inline-block werden Elemente nebeneinander angezeigt.
font-size Mit dieser Eigenschaft stellen Sie die Größe der verwendeten Schrift ein. Die Standardgröße ist 16px. Die Schriftgröße kann als numersicher Wert oder als relativer Wert angegeben werden.
cursor Mit dieser Eigenschaft können Sie das Aussehen des Mauszeiges ändern, wenn dieser auf ein bestimmtes HTML-Element zeigt. Hierbei gibt es viele möglichen Werte, die angenommen werden können.
.button{
background-color: red;
border: none;
color: whith;
padding: 15px 32px;
text-align: center;
text-decoration: none;
display: inline-block;
font-size: 16px;
cursor: pointer;
}
